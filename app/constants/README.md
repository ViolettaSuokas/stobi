# `constants/` — дизайн-токены и статичные значения

Единый источник правды для визуальных констант. Не клади сюда бизнес-конфиги.

## Файлы

| Файл | Что это |
|---|---|
| `Colors.ts` | Палитра (bg, accent, text, green, border и т.д.) |
| `Spacing.ts` | 4-based шкала отступов (`xxs=2 ... huge=40`) + Radius |
| `Typography.ts` | Шкала текста: `display`, `heading1-3`, `body`, `caption`, `button`, `label` |

## Convention

- Всё — `export const` с `as const` (неизменяемо на уровне типов)
- Никаких функций и хуков — только данные
- Ни один компонент не должен хардкодить hex/magic-numbers — бери токен

## Как добавить новый цвет / отступ / шрифт

1. Добавь ключ в соответствующий файл
2. Используй: `import { Colors } from '../../constants/Colors'` → `Colors.newThing`
3. **Никогда** не копируй hex/размеры между файлами
