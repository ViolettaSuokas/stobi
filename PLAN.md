# Stobi — План разработки

> Мобильное приложение для любителей расписных камней в Финляндии  
> Стек: **React Native + Expo** (iOS + Android из одного кода) · **Supabase** (backend) · **Figma** (дизайн)

## Фаза 3 — Геймификация (Камень-аватар)

### 3.1 Система аватара
- [ ] Базовый цифровой камень у каждого пользователя
- [ ] Встроенный редактор — рисуй пальцем на камне
- [ ] Слои: фон, узоры, детали, эффекты
- [ ] Анимация "живого" камня (покачивание, блеск)

### 3.2 Система опыта
- [ ] XP за действия:
  - Спрятал камень → +10 XP
  - Нашёл камень → +15 XP
  - Получил лайк на камень → +5 XP
  - Ежедневный вход → +2 XP
- [ ] Уровни (1-50) → разблокируют новые элементы декора
- [ ] Краски и паттерны как награды

### 3.3 Коллекции и достижения
- [ ] Бейджи за достижения (первый камень, 10 находок, и т.д.)
- [ ] Коллекция найденных камней в профиле
- [ ] Рейтинг активных пользователей (по городам)

## 🛠️ Технический стек

| Компонент | Технология | Почему |
|---|---|---|
| Мобильное приложение | React Native + Expo | iOS + Android из одного кода |
| Навигация | Expo Router | файловая навигация, удобно |
| Стили | NativeWind (Tailwind) | быстро, консистентно |
| Карта | react-native-maps | зрелая, надёжная |
| Backend | Supabase | auth + база + storage + real-time из коробки |
| База данных | PostgreSQL (Supabase) | геолокация через PostGIS |
| Хранилище фото | Supabase Storage | просто, дёшево |
| Push уведомления | Expo Notifications | нативные уведомления |
| Подписки | RevenueCat | In-App Purchase для iOS и Android |
| Платежи (Artist) | Stripe | стандарт |
| Аналитика | Mixpanel или PostHog | трекинг событий |
| Краши | Sentry | отлавливать ошибки |

## 📊 База данных (схема)

```sql
-- Пользователи
users: id, username, avatar_url, bio, level, xp, is_premium, is_artist

-- Камни
stones: id, user_id, photo_url, lat, lng, description, tags[], is_hidden, found_count

-- Находки
finds: id, stone_id, finder_id, photo_url, found_at, re_hidden_lat, re_hidden_lng

-- Аватар камень
stone_avatars: id, user_id, layers_json, skin_id, last_updated

-- Достижения
achievements: id, user_id, badge_type, earned_at
```
