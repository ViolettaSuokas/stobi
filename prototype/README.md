# prototype/ — LEGACY HTML-прототип

Это **НЕ** рабочий код. Это ранний HTML/CSS/JS-прототип Stobi
(когда ещё не было Expo-приложения).

## Для чего хранится

- Reference для дизайнера/PM — как выглядела начальная идея
- Быстрый способ показать концепт без установки Expo Go
- Архив решений по UX которые потом перекочевали в RN

## Для чего НЕ используется

- Продакшен (никогда не деплоится)
- Shared code между web и mobile (app/ самостоятельный)
- Актуальные UI-решения — смотри `app/` (мобильное приложение)

## Безопасно удалить?

Да — ничего в рабочем коде не импортирует отсюда. Но пока оставь,
он иногда пригождается для питча / маркетинга. Весит мало (~50 kB).
